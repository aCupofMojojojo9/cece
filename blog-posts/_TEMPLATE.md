# =============================================

# HOW TO ADD A NEW BLOG POST

# =============================================

#

# 1. COPY this file and rename it using this format:

# YYYY-MM-DD-short-title.md

# Example: 2026-03-20-liberia-trip-update.md

#

# The date in the filename controls the sort order

# (newest date = shows first on the page).

#

# 2. FILL IN the details between the --- lines below.

#

# 3. WRITE your post content after the second --- line.

#

# 4. Save the file in this blog-posts/ folder and

# rebuild/redeploy the site. Your post will appear!

#

# =============================================

# FORMATTING TIPS

# =============================================

#

# ## Heading <- Big section heading

# ### Subheading <- Smaller subheading

#

# **bold text** <- Makes text bold

# _italic text_ <- Makes text italic

#

# - Item one <- Bullet list

# - Item two

# - Item three

#

# Just leave a blank line between paragraphs.

#

# =============================================
